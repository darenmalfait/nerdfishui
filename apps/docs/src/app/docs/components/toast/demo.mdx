## Usage

### Toaster

This needs to be placed in the root component of your application.

```tsx
import { Toaster } from '@nerdfish/ui'
```

```tsx
<Toaster />
```

### Toast

```tsx
import { Button, toast } from '@nerdfish/ui'
```

```tsx
<Button onClick={() => toast.message('Toast message')}>Show toast</Toast>
```
