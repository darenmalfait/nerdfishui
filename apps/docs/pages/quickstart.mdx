# Quickstart

To install the component-library run the following within your project
directory.

<Code
  language="bash"
  code={`npm i @nerdfish/ui-core @nerdfish/ui postcss tailwindcss @tailwindcss/typography @heroicons/react`}
/>

## Configuration

The theme of this library depends on the @tailwindcss/typography plugin. To use
it, follow the steps on the plugin source page.
https://github.com/tailwindlabs/tailwindcss-typography

<Code
  language="javascript"
  code={`//tailwind.config.js
module.exports = {
  mode: 'jit',
  content: [
    // ... paths that use tailwind
    './node_modules/@nerdfish/**/*.{js,ts,jsx,tsx}', // path to nerdfishui
  ],
  theme: {
    extend: {},
  },
  plugins: [require('@tailwindcss/typography'), require('@nerdfish/ui-core')], 
}`}
/>

<Code
  language="javascript"
  code={`//postcss.config.js
module.exports = {
    plugins: {
      tailwindcss: {},
      autoprefixer: {},
    },
}`}
/>

Then you need a global css file which you import at the root of the project

<Code
  language="javascript"
  code={`//styles.css
@tailwind base;
@tailwind components;
@tailwind utilities;`}
/>

## Usage

<Code
  language="javascript"
  code={`import * as React from 'react'
import {ThemeProvider, H1} from '@nerdfish/ui'
import '@nerdfish/theme/dist/nerdfishui.css'

function App() { return ( <ThemeProvider> <H1>Hello Nerdfish</H1>

</ThemeProvider> ) }`} />
