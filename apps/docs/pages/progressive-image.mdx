# ProgressiveImage

PrograssiveImage is an image component that loads the image progressively with a
placeholder.

<Code code={`import { ProgressiveImage } from '@nerdfish/ui'`} />

## Properties

<Properties>
  <Property name="placeholder" type="string">
    The placeholder image url.
  </Property>
  <Property name="img" type="React.ReactNode">
    A node that will be rendered as the image.
  </Property>
</Properties>
