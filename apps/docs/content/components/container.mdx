## Usage

Contaienr is a child of the [Grid](/grid) component. It is used to wrap content
and set the width of the content.

```tsx
import {Grid, Container} from '@nerdfish/ui'
```

```tsx
<Grid nested>
  <Container>
    <div>Hello world</div>
    <div>Hello world</div>
  </Container>
</Grid>
```
