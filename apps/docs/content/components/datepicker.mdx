## Usage

Datepicker extends the [Input](/input) component. All props from Input are
available.

```tsx
import {Datepicker} from '@nerdfish/ui'
```

```tsx
<Datepicker
  label="datepicker label"
  description="datepicker description"
  name="datepicker-field-basic"
/>
```
