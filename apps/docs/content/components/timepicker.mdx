## Usage

Timepicker extends the [Input](/input) component. All props from Input are
available.

```tsx
import { Timepicker } from '@nerdfish/ui'
```

```tsx
<Timepicker
	label="timepicker label"
	description="timepicker description"
	name="timepicker-field-basic"
/>
```
